# Epic 4: Laravel Project Experience

This package is the focused execution plan for Epic 4 of the pv rewrite.

Epic 4 makes the Laravel path first-class: project contracts, `pv init`,
`pv link`, managed `.env` writes, setup commands, HTTPS `.test` routing,
`pv open`, and Laravel helper commands.

## Documents

- `project-plan.md` - Epic 4 work hierarchy, dependencies, risks, and done rules.
- `implementation-plan.md` - implementation sequence and package guidance.
- `issues-checklist.md` - GitHub issue bodies and publishing tracker.
- `test-strategy.md` - focused test strategy for Laravel project workflows.
- `qa-plan.md` - review gates and manual QA expectations.
